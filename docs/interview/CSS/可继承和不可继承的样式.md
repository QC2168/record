## 可继承和不可继承的样式

### 可继承的样式

- 字体
  - font-family
  - font-weight
  - font-size
  - fonts-style
- 文本
  - text-indent
  - text-align
  - line-height
  - word-spacing
  - letter-spacing
  - color
  - text-transform
- 可见
  - visibility
- 列表
  - list-style
  - list-style-type
  - list-style-image
- 光标
  - cursor


### 不可继承的样式

- 文本属性
  - display 元素类型
  - vertical-align：垂直文本对齐
  - text-decoration：规定添加到文本的装饰
  - text-shadow：文本阴影效果
  - white-space：空白符的处理
- 盒模型属性
  - width
  - height
  - margin
  - padding
  - border
- 背景属性
  - background
  - background-image
  - background-color
  - background-repeat、position、attachment
- 定位属性
  - float
  - clear
  - position（left，right，top，bottom）
  - overflow
  - clip
  - z-index
- 轮廓属性
  - outline
  - outline-style
  - outline-weight
  - outline-color
- 页面属性
  - size
  - page-break-before
  - page-break-after
- 声音属性
  - pause-before
  - pause-after
  - pause
  - cue-before
  - cue-after
  - cue
  - play-during
